# Plan Groups

Plan groups allow grouping related plans so customers can make **upgrades and downgrades** from the Customer Portal.

## Business Rules

- Customers can only switch between plans from the same group
- Plan order determines upgrade vs downgrade
- All plans in a group **must support the same set of regional currencies**
- Plans not in any group: customer cannot change from portal (operator only)

## What Happens on Change

| Change Type | Behavior |
|-------------|----------|
| **Upgrade** | Immediate, prorated charge |
| **Downgrade** | At end of current period |
